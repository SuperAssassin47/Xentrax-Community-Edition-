# Xentrax-Community-Edition-
Xentrax CE is a fully autonomous AI-powered live network sniffer that sniffs the network suspicious packets, blocks them and logs them if needed.

# How to use Xentrax in terms of AI Development

Xentrax CE is a neural engine designed to run on its own. Upon startup, it takes around 3-5 seconds for the engine to start and build Neural Network. Once built, it will initiate the live network sniffer and begin sniffing.
