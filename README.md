# Xentrax-Community-Edition-
Xentrax CE is a neural AI engine built using Python and Flask for turning the engine itself into a local server API. It runs constantly and idles so it doesn't hassle you when you decide build your own chatbot and integrating the engine.

# How to use Xentrax in terms of AI Development

Xentrax CE is a neural engine designed to run on its own. Upon startup, it takes around 3-5 seconds for the engine to start and build Neural Network. Once built, it will automatically initiate a training algorithm which trains the AI Entity so it's ready for connecting your chatbot.

# Future Optimizations

In future updates, I will plan on evolving the Neural Network and introducing more neurons, more layers and making it better for all of you. It's currently a simple engine with a communication layer which idles whilst waiting for a chatbot to connect to it. Enjoy! :)
